# 🐾🌸 Welcome to MeowTopia! 🎀🐱✨

Hello, gorgeous! 💖 Welcome to **MeowTopia** — the most purr-fect, cutest little corner of the internet where you can fall in love with and adopt sweet, cuddly kitty-cats! 🐈🎀

This magical project is split into two cozy homes: one for our beautiful **Frontend** interface, and one for our hardworking **Backend** server. Let’s take a look around! 🥰🌷

---

## 🎀 Directory Tour 🐾

### 🌸 `Frontend/` (The Pretty Face 💅)
Built with **React ⚛️ + Vite ⚡**, this is where the magic happens! It's decorated with pastel colors, micro-animations, and cute cat illustrations.
* 🐾 **Favorites Context:** Keeps track of all the precious kitties you've starred! ⭐
* 🐾 **Adoption Context:** Manages the sweet cats you have chosen to bring home forever! 🏡💖
* 🐾 **API Helpers:** Communicates directly with our backend so no kitty's love is ever lost!

### 🌸 `Backend/` (The Purring Engine ⚙️🐱)
Built with **Node.js 🟢, Express 🚂, and MongoDB 🍃**, this is where all our user details and kitty records are kept safe and warm.
* 🐾 **Direct User Relationships:** Your favorite cats and adoptions are saved directly in your user profile! No messy separate databases—everything is neat and tidy like a freshly groomed kitty! 🧼✨
* 🐾 **Security:** Hashed passwords and secure JWT tokens keep the bad dogs away! 🐶🚫

## 🚀 How to Start Your Meow-tastic Journey 🌸

Follow these simple steps to get both servers running!

### 1️⃣ Set up the Backend (The Brains 🧠✨)
Open your terminal, go to the backend folder, and install the goodies:
```bash
cd Backend
npm install
```

Create a cozy `.env` file inside the `Backend/` folder (we already set one up for you!):
```env
PORT=5000
MONGO_URI=your_mongodb_atlas_connection_uri_here
JWT_SECRET=some_super_secret_cute_key
JWT_EXPIRES_IN=7d
```

Start the server and watch it purr:
```bash
npm run dev
```
*The server will run on `http://localhost:5000` 🎈*

### 2️⃣ Set up the Frontend (The Beauty 🎀✨)
Open another terminal, go to the frontend folder, and install the treats:
```bash
cd Frontend
npm install
```

Start the Vite development server:
```bash
npm run dev
```
*Open `http://localhost:5173` in your browser and get ready to go "Awww!" 😍🌸*
